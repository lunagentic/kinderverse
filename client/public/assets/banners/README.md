# 섹션 제목 배너 (Title Banner Layer)

킨더랩 레이아웃에서 각 섹션 제목 자리에 쓰는 **리본 배너 이미지**.
아래 파일명 그대로 이 폴더(`client/public/assets/banners/`)에 넣어주세요.
파일이 없으면 자동으로 기존 텍스트 제목으로 표시됩니다(폴백).

## 권장 사양
- 형식: PNG (배경 투명)
- 가로:세로 비율 ≈ **5:1 ~ 6:1** (가로로 긴 리본)
- 권장 크기: 가로 700px 이상

## 파일명 ↔ 섹션
| 파일명 | 섹션 |
|---|---|
| `reason.png` | 놀이 선정 이유 |
| `flow.png` | 이 달의 놀이 흐름 |
| `expectation.png` | 교사의 기대 |
| `curriculum.png` | 교육과정 연계 |
| `goal.png` | 놀이 목표 (선택) |
| `environment.png` | 놀이 환경 구성 (선택) |
| `outdoor.png` | 바깥놀이·신체활동 (선택) |
| `safety.png` | 안전교육 (선택) |
| `character.png` | 인성교육 (선택) |
| `home.png` | 가정 연계 (선택) |
| `event.png` | 행사 (선택) |

> 보내준 5종(놀이 선정 이유 / 이 달의 놀이 흐름 / 놀이 목표 / 교사의 기대 / 놀이 환경 구성)은
> 각각 `reason.png`, `flow.png`, `goal.png`, `expectation.png`, `environment.png` 로 저장하면 됩니다.
