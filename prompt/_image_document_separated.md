# _image_document.md

## 목적
구조화된 놀이계획 데이터를 PNG 문서로 렌더링한다.

## 지원 문서
- 주안
- 월안
- 일안
- 프로젝트계획안
- 프로젝트안내문
- 놀이미션카드

## 렌더링 파이프라인

PLAN JSON
→ normalize
→ template
→ SVG Layer
→ PNG

## 놀이 활동 시각화 규칙

### 기본 원칙
활동명
→ 놀이 행동
→ 놀이 결과

### 시각화 우선순위
1. 유아 놀이 장면
2. 놀이 결과물
3. 놀이 재료
4. 아이콘

### 필수 구성
유아 + 행동 + 결과물

### 금지
- 재료만 표현
- 아이콘만 표현
- 결과물만 표현

## 문서별 레이아웃
각 문서 타입은 독립 템플릿 사용

## 콘텐츠 정규화
제목, 카드제목, 불릿 수 제한 적용

## 품질 체크
- 문서 타입 일치
- 정보 계층 명확
- 놀이 장면 중심
- PNG 즉시 활용 가능
